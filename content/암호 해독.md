## 암호해독!

###### 문제 설명

모든 알고리즘을 해독할 수 있는 알고리즘 7 원석를 보유한 알고리즘 제왕 **파이**와 **썬**은 죽기 전, 이 보물에 '암호'를 걸어 세계 어딘가에 묻어놨다고 공표하였다. 그가 남긴 문자는 아래와 같다.

##### 제한사항

##### 입출력 예

| data                                                          | return |
| ------------------------------------------------------------- | ------ |
| [ + -- + - + - , + --- + - + , + -- + - + - , + - + - + - + ] | JEJU   |

##### 입출력 예 설명

입출력 예#1
data를 받아서 JEJU를 출력합니다.

```javascript
function solution(data) {
  let answer = "";

  answer += String.fromCharCode(
    parseInt(
      data
        .replace(/ /g, "")
        .replace(/\+/g, "1")
        .replace(/-/g, "0"),
      2
    )
  );
}
```

# Python

### powershell명령

- ls : 현재 디렉토리(폴더)의 내용을 확인하는 명령
- cd : powershell에서 련재 디렉토리(폴더)를 바꾸는 명령

### print() 함수

- 괄호 안에 있는 내용을 화면에 출력하는 명령
- 화면에 표시하고 싶은 글은 따옴표(')로 묶어서 표현

```python
print('Hello world!!')
```

### 함수란?

- 파이썬의 특정한 기능(예, 화면에 출력)을 수행하도록 하는 명령
